Tag - Element - Attribute

Tag

Which comes in < Angularbrackets >

for eg, h1, p, meta, title etc.

Tag are of two types

  1. Paired tags -> Which comes with start tag (Opening Tag) and End Tag (Close Tag) for eg. h1, body, head, p, etc
  2. Self Closing Tag -> Which have only starting tag , they don't need any closing tag, They are also called void Tag and Empty Tag. for eg. br, hr, img, input, meta etc

Element

Start Tag - Some Content - End Tag

Attriburte

They provide some Additional Information by which behaviour of Element Change, They always written in start tag
